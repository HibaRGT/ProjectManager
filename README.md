# Agile Project Management API

This is a **RESTful API** for managing projects following the **Agile methodology**.  
It allows teams to create and manage **Product Backlogs, User Stories, Epics, and Sprints**.  

The API handles authentication, and secure access using **JWT**.

---

## Features
- **Role based access**
- Create and manage **Product Backlogs**  
- Define and organize **Epics and User Stories**  
- Plan and track **Sprints**  
- Store data in **MySQL**

---

## Tech Stack
- **Backend:** Spring Boot 
- **Database:** MySQL  
- **Other Tools:** Maven, Lombok, JPA/Hibernate, Swagger
